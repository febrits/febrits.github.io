# Muhammad Febrilian Tisna | Portfolio 2026

Personal portfolio website showcasing projects, skills, and experience.

- React 19 + TypeScript
- Vite
- Tailwind CSS v4
- Framer Motion

## Development

**Prerequisites:** Node.js 18+

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
